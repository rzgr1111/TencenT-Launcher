# Minecraft Launcher

Modern ve özelleştirilebilir bir Minecraft launcher.

## Özellikler

- 🎮 Tüm Minecraft versiyonları desteği
- 👤 Microsoft ve Mojang hesap girişi
- 📦 Profil yönetimi
- 🔧 Mod desteği
- 🎨 Modern ve kullanıcı dostu arayüz
- 🔒 Güvenli hesap yönetimi

## Kurulum

```bash
# Bağımlılıkları yükle
pip install -r requirements.txt

# Launcher'ı başlat
python src/main.py
```

## Geliştirme

Bu proje Python ve CustomTkinter kullanılarak geliştirilmiştir.

## Güvenlik

⚠️ **ÖNEMLİ**: Hesap bilgileriniz ve tokenlarınız yerel olarak şifrelenmiş şekilde saklanır. 
Asla `data/` klasörünü paylaşmayın veya git'e eklemeyin.
