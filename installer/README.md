# TencenT Launcher - Installer

## Installer'ı Derlemek

```bash
# Bağımlılıkları yükle
pip install -r requirements.txt
pip install pyinstaller

# Installer'ı derle
python build_installer.py
```

## Çıktı

`dist/TencenT-Launcher-Installer.exe` dosyası oluşacak.

## Özellikler

- ✨ Animasyonlu açılış ekranı
- 📥 Otomatik Java kurulumu
- 💾 Özelleştirilebilir kurulum yolu
- 🖥️ Masaüstü kısayolu
- ✅ Zaten kurulu kontrolü
